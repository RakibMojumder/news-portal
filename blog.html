    What is the difference between var, let and const ?
    • 1. Var declarations are globally scoped or function scoped. Let & Const are Blockd scope;
    • 2. Var can be redeclared. But Let & Const can not be redeclared.
    • 3. Var & Let can be reassigned. But Const can not be reassigned.
    • 4. Var and Const can be declared without being initialized Const must be initialized during declaration.
    What is the difference between arrow function and regular function ?
    • 1. In regular function there is an argument local variable. But, in arrow function threre is no arguments;
    • 2. return keyword can be omit if the arrow function has only one line. But in regular function you have to use return keyword for return something from a function.
    • 3. regular function allowed duplicate parameter name. But in arrow function can not take duplicate parameter name.
    What is the difference among map, forEach, filter and find ?
    • 1. forEach() : forEach function has called for all the array single element. It's using for looping in an array.
    • 2. map() : map function is almost similar to forEach. The difference is forEach function don't need return anything. But in map it has called for all the array single element and return an array.
    • 3. filter() : filter method use to filtering an array based on a condition and it's return an output in an array.
    • 4. find() : find method is similar to filter method. The difference is find method return only first match element based on condition.
